# IAM Users, Groups & Policies Lab

## Objective
To create IAM users, groups, and attach policies to manage secure access in AWS.

---

## What I did

### 1. Created IAM Users
- Created user: cloud-user1
- Enabled AWS Management Console access

### 2. Created IAM Group
- Group name: CloudAdmins
- Attached policy: AdministratorAccess

### 3. Added User to Group
- Added cloud-user1 to CloudAdmins group

---

## Result
- User inherited permissions from group
- Verified access successfully

---

## Key Learning
- IAM groups made permission management easier
- Policies control access to AWS resources
